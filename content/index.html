Elasticity Tool

Interactive tool for calculating price elasticity and cross-price elasticity for soda products using real market data.

Open in JupyterLab Open in Classic Notebook
Note: This runs entirely in your browser using WebAssembly. No server required! The first load may take a moment as Python packages are downloaded.

Features

  • Calculate own-price elasticity for specific products
  • Analyze cross-price elasticity between products
  • Interactive visualizations with real-time updates
  • Market analysis across multiple brands
  • Works offline once loaded
